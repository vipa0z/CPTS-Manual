## 01. Asset Discovery
- [ ] **Subdomain Discovery** ‚Üí `amass`, `subfinder`, `assetfinder`
- [ ] **VHost Discovery** ‚Üí `ffuf`/`gobuster` with `Host:` header fuzzing
- [ ] **Directory Bruteforce** ‚Üí `ffuf`, `dirsearch`, `gobuster`
- [ ] **Parameter Discovery** ‚Üí `arjun`, wordlist-based fuzzing
- [ ] **Archived Content Search** ‚Üí Wayback Machine, `waymore`

---

## 02. Mapping & Crawling

- [ ] **Automated Mapping** ‚Üí BurpSuite Pro crawler/spider
- [ ] **Manual Mapping** ‚Üí Browse key functions, note parameter flows
- [ ] **Identify Entry Points** ‚Üí Forms, file uploads, API endpoints

---

## 03. Server & Framework Fingerprinting
- [ ] **Technology Stack Analysis** ‚Üí `wappalyzer`, `whatweb`, `builtwith`, HTTP headers inspection (`httpx`, Burp)
- [ ] **Web Server Version Check** ‚Üí Apache/Nginx/IIS version in headers
- [ ] **Framework Detection** ‚Üí Laravel, Django, Express, Rails, etc.
- [ ] **Identify Middleware / WAF** ‚Üí `wafw00f`, HTTP response patterns

## 05. HTTP Method / Verb Testing
- [ ] **Auth Page Verb Tampering** ‚Üí Test `GET`, `POST`, `PUT`, `DELETE`, `OPTIONS`
- [ ] **Verb Tampering on Parameterized Requests** ‚Üí Try alternate verbs, override headers (`X-HTTP-Method-Override`)

---

## 06. Fuzzing & Automation
- [ ] **Directory Fuzzing** ‚Üí `ffuf`/`gobuster` on `/` and discovered paths
- [ ] **Parameter Fuzzing** ‚Üí `ffuf` with GET/POST param lists
- [ ] **Burp Pro Active Scan**
- [ ] **Custom Payload Lists** ‚Üí For tech-specific injections

---

## 07. Business Logic & Access Control Testing
- [ ] Test role escalation (user ‚Üí admin)
- [ ] Test rate-limiting bypasses
- [ ] Test multi-step workflows for bypasses (e.g., skipping payment)
